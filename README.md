<h2>其它</h2>

本来笔者是想把上述文献中的观点，消化吸收后形成自己的文件写下来的。可是，大师（Bjarne Stroustrup等）的观点总是鞭辟入里，高屋建瓴。这是笔者所不能及的，自己"蛮横"的改动，可能会曲解大师的意图和观点。

所以，遵循不重复造轮子的原则，大部分文字都是笔者直接拷贝的，但有时也会有笔者自己的观点。
